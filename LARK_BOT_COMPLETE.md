## 🚀 快速开始（3种方式）

### 方式1：本地测试（5分钟）
```bash
cd lark-bot
npm install
npm run setup    # 配置向导
npm run dev      # 启动服务
```

### 方式2：云服务器部署（30分钟）
```bash
# 购买云服务器（阿里云/腾讯云）
# 配置：1核2G，1M带宽，Ubuntu 22.04

# 一键部署
cd lark-bot
./scripts/deploy.sh 47.77.237.222 root

# SSH配置
ssh root@47.77.237.222
cd /opt/lark-bot
nano .env       # 填写飞书凭证
./start.sh      # 启动服务
```

### 方式3：Docker部署
```bash
cd lark-bot
docker-compose up -d
```
