## 快速命令参考

### 本地开发
```bash
# 进入目录
cd lark-bot

# 安装依赖
npm install

# 配置向导
npm run setup

# 启动开发服务器
npm run dev
```

### 服务器部署
```bash
# 一键部署到服务器
./scripts/deploy.sh 47.77.237.222 root

# SSH 登录服务器配置
ssh root@47.77.237.222
cd /opt/lark-bot
nano .env  # 编辑配置
./start.sh  # 启动服务
```

### 常用 PM2 命令
```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs
pm2 logs --lines 100

# 重启服务
pm2 restart lark-bot

# 停止服务
pm2 stop lark-bot

# 开机自启
pm2 save
pm2 startup
```

## 故障排查

### 问题1：飞书验证失败
- 检查服务器是否可以被公网访问
- 检查端口 3000 是否开放
- 查看应用日志：`pm2 logs`

### 问题2：机器人不响应消息
- 确认机器人已添加到群聊
- 检查是否正确 @机器人
- 查看日志是否有收到消息事件

### 问题3：部署脚本失败
- 检查 SSH 连接是否正常
- 检查服务器是否有 Node.js 环境
- 手动检查错误信息
